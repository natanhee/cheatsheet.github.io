CSS Colors

Name Description Exemple
Hexadecimal colors hexadecimal color is specified with: #RRGGBB #00ff00
RGB colors An RGB color value is specified with the rgb() function rgb(255, 0, 0)
HSL colors HSL color value is specified with the hsl() function hsl(120, 100%, 50%)

CSS Colors with Transparancy

Name Description Exemple
Hexadecimal colors To add transparency, add two additional digits between 00 and FF #ff000080
RGBA Colors An RGBA color is specified with the rgba() function rgba(0, 0, 255, 0.3)
HSLA colors An HSLA color value is specified with the hsla() function hsla(120, 100%, 50%, 0.5)
